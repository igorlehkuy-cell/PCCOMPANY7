# PC Company - Веб-сайт для продажу комп'ютерів та ноутбуків

## Структура проекту

```
PC/
├── Frontend/
│   ├── html/
│   │   ├── index.html (головна сторінка)
│   │   ├── login.html (сторінка входу)
│   │   ├── register.html (сторінка реєстрації)
│   │   ├── rental.html (сторінка оренди)
│   │   ├── customization.html (сторінка кастомізації)
│   │   ├── cart.html (кошик)
│   │   └── checkout.html (оформлення замовлення)
│   ├── CSS/
│   │   ├── styles.css (основні стилі)
│   │   ├── auth.css (стилі для login/register)
│   │   ├── cart.css (стилі для кошика)
│   │   ├── checkout.css (стилі для оформлення)
│   │   ├── rental.css (стилі для оренди)
│   │   └── customization.css (стилі для кастомізації)
│   └── JS/
│       ├── script.js (основний скрипт з модальним вікном)
│       ├── cart.js (управління кошиком)
│       ├── checkout.js (оформлення замовлення)
│       ├── rental.js (сторінка оренди)
│       └── customization.js (сторінка кастомізації)
└── package.json
```

## Функціональність

### ✅ Реалізовано:

1. **Головна сторінка (Index)**
   - Компактний хедер з навігацією
   - Логотип PC◗Company
   - Кнопки входу та реєстрації
   - Соціальні мережі (Instagram, Facebook, YouTube)
   - Кнопка кошика
   - Каталог товарів з модальним вікном
   - Розділ "Чому обирають нас?"
   - Розділ спонсорів
   - Розділ "Покупцям" (гарантія, умови оплати)
   - Контакти

2. **Модальне вікно товару**
   - Характеристики товару
   - Ціна
   - Система відгуків з можливістю додання
   - Кнопка "Додати в кошик"
   - Плавні анімації

3. **Вхід та Реєстрація**
   - Форма входу з email та паролем
   - Форма реєстрації з валідацією
   - Зберігання даних у localStorage
   - Коректна навігація

4. **Кошик**
   - Перегляд товарів у кошику
   - Управління кількістю
   - Видалення товарів
   - Сума замовлення
   - Кнопка оформлення замовлення

5. **Оформлення замовлення**
   - Персональна інформація
   - Вибір міста доставки з цінами
   - Вибір способу оплати (карта, переказ, готівка, крипто)
   - Збереження замовлення
   - Попередній перегляд замовлення

6. **Окремі сторінки**
   - Оренда комп'ютерів та ноутбуків
   - Кастомізація комп'ютерів
   - Правильна навігація між сторінками

7. **Дизайн та UX**
   - Responsiv дизайн для всіх екранів
   - Плавні анімації (spring, rotate, scale)
   - Темні та світлі елементи UI
   - Коричнева та зелена колірна схема
   - Українська мова

### 🎨 Анімації:

- Хедер: соціальні іконки та кошик з spring анімаціями
- Модальне вікно: слайд-вверх анімація
- Кнопки: hover ефекти з масштабуванням та обертанням
- Товари: fade-in при прокручуванні

## Запуск проекту

1. Відкрити файл `Frontend/index.html` у браузері
2. Перейти по посиланням у навігації
3. Тестувати функціональність кошика та оформлення

## Локальне зберігання (localStorage)

- `pc-company-user` - дані користувача після реєстрації
- `pc-company-registered` - статус реєстрації
- `pc-company-logged-in` - статус входу
- `pc-company-current-user` - поточний користувач
- `pc-company-cart` - товари в кошику
- `pc-company-orders` - історія замовлень

## Технологічний стек

- HTML5
- CSS3 (з анімаціями та grid/flex)
- Vanilla JavaScript (без фреймворків)
- localStorage API
